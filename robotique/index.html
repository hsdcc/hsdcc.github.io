Résumé Robotique

Séance 1 + 2

La Robotique et sa Définition

Robotique : Science mécatronique qui s'intéresse aux robots
Mécanique + Électronique + Informatique = Mécatronique

Les Robots

Robot : Machine capable d'effectuer des tâches et de manipuler des objets selon un programme

Utilisations

Situations dangereuses

Haute précision

Tâches répétitives

Situations où humains sont incapables

Composition du Robot

Capteurs

Détectent les phénomènes physiques comme la température, la lumière, la distance, etc.

Actionneurs

Dispositifs permettant d'effectuer des actions mécaniques, électriques, ou optiques

Source d'alimentation

Fournit l'énergie nécessaire pour alimenter le robot

Microcontrôleur

Le cerveau du robot qui traite les informations et contrôle les périphériques

Fonctionnement du Microcontrôleur

Traitement des informations

Entrée

Récupération des informations

Sortie

Contrôle des périphériques

L'Électricité

Électricité : Forme d'énergie résultant du mouvement des électrons dans un conducteur

Électron : Particule chargée négativement qui se déplace dans un circuit électrique

Courant électrique : Flux d'électrons qui circulent du pôle négatif vers le pôle positif

La Programmation

Programmation : Langage formel pour communiquer avec une machine

Les 3 Étapes

1

L'Algorithme

Pseudocode ou organigramme

SI température > 25°
  ALORS allumer ventilateur
SINON
  ALORS éteindre ventilateur
↓
2

Le Codage

Transformer l'algorithme en langage de programmation

if (temperature > 25) {
  fan.turnOn();
} else {
  fan.turnOff();
}
↓

Compilation

Transformer tout le code en langage machine avant l'exécution

// Code C
#include <stdio.h>

int main() {
    printf("Hello World!");
    return 0;
}
↓
// Compilation
gcc hello.c -o hello
./hello

Interprétation

Exécuter le code ligne par ligne sans compilation préalable

# Code Python
print("Hello World!")
↓
# Exécution directe
python hello.py
